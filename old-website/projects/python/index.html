fizzBuzz

A program that generates number from 1 to 100, but all numbers that can be divided by 3 with 0 remainder will generate the world "fizz", while if the numbers can be divided by 5 with 0 remainder will generate the word "buzz", if the numbers can be divided by 5 nor 3 with 0 remainder,then the console will generate the world "fizzBuzz". Frequently used for coding interviews.

Download right here.

Note: Use Python 3 Interpreter for running this program. By opening command prompt/terminal in the folder that contains the python script file (fizz_buzz.py), the run the command "python3 fizz_buzz.py" . NOTE: Without the quotation marks!
